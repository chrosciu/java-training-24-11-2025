1. Utworz klase lub rekord User z polami: String firstName, String lastName
2. Utworz klase UserStorage z metoda: public static List<User> getAll()
Metoda ta powinna zwracac kilku uzytkownikow (np. 3) w formie listy.
3. Utworz klase UserServlet, ktora powinna po uderzeniu HTTP pod adres /users zwracac liste uzytkownikow jako odpowiedz HTTP.
Klasa ta powinna wykorzystywac metode UserStorage.getAll() do pobrania listy uzytkownikow.
Format moze byc dowolny (zwykly tekst, JSON itp.)
4. Uruchomic serwer i sprawdzic w przegladarce czy endpoint dziala poprawnie.
